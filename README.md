# Xoriva — Backend API

Built with **Node.js**, **Express.js**, **MongoDB**, **JWT**, **Bcrypt**, **Stripe**, and **Google Gemini API**.

## Features

- RESTful API architecture with role-based access control for Admin and User roles
- JWT authentication with bcrypt password hashing for secure, stateless sessions
- Stripe payment gateway integration with payment intent creation and webhook verification
- Google Gemini AI API integration powering the intelligent customer support chatbot
- Product and order management with full CRUD operations and status lifecycle tracking
- Peer-to-peer thrift marketplace backend — listing creation, approval flow, and buyer/seller handling
- MongoDB + Mongoose for flexible, schema-driven data modeling
- Rate limiting and CORS configured for production-level API security
